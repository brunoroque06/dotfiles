# Dots

## Installation

```shell
./dots
```

Supported: macOS.

## Manual Steps

- Setup `Colemak` keyboard layout;
- Remap `Caps Lock` to `Control` (HHKB);
- Load `Raycast` configuration.

## Shortcuts

- [GNU Readline](https://en.wikipedia.org/wiki/GNU_Readline);
- [macOS](https://support.apple.com/en-us/HT201236).
