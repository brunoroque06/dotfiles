# Dots

## Installation

```shell
./dots
```

Supported: macOS.

## Manual Steps

- Setup `Colemak` keyboard layout;
- Remap `caps lock` to `control`;
- Load `Raycast` configuration;
- Configure `rcmd`: `right control` (`caps lock`) + `right shift`.
