# Dots

## Installation

```shell
./dots
```

Supported: macOS.

## Manual Steps

- Setup `Colemak` keyboard layout;
- Remap `Caps Lock` to `Control` (HHKB);
- Load `Raycast` configuration;
- Configure `rcmd` (`lctrl` + `rshift`).
